
# Magasinet Filter

A Progressive Web App made for Magasinet Filter, using [react](https://reactjs.org/).


### Setup
- Download or clone the project
- Navigate with the command prompt to the root file of the project
- Run commands:
```sh
  $ npm install
  $ npm start
```
